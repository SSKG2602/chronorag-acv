# Layer 2 Chart Assets

This folder is reserved for public or LinkedIn-ready chart specifications after
real Layer 2 benchmark results exist.

Do not create charts with invented values. Use generated result JSON files from
`benchmarks/layer2_crossdomain/results/` only.

Planned chart types:

- corpus scale by domain
- method comparison grouped bar chart
- valid-time accuracy comparison
- evidence grounding comparison
- failure category breakdown
